# NEXUS ERP CRM HR AI Platform

Enterprise modular platform for ERP, CRM, HRM, Recruitment, Finance, Inventory, Analytics, AI automation and realtime monitoring.

## Stack

- Frontend: Next.js 15, React, TypeScript, Tailwind CSS style system
- Backend: Node.js, NestJS, REST API, RBAC-ready guards
- Data layer: PostgreSQL-ready module boundaries, Redis/Elasticsearch integration points
- DevOps: Docker Compose baseline, CI/CD-ready monorepo

## Quick Start

```bash
npm install
npm run dev
npm run dev:api
```

Web app: `http://localhost:3000`

API: `http://localhost:4000/api`

## Workspace

```text
apps/web      Premium enterprise dashboard
apps/api      NestJS modular backend
packages/shared  Shared roles, domain types and constants
```
